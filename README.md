# Klekovina - Hiking Trip Organizer

A beautiful web app for creating and organizing hiking trips. Built with SvelteKit and Tailwind CSS.

## Features

- 🏔️ Create and manage hiking trips with detailed information
- 📍 Organize trips by location, difficulty level, and date
- 🎨 Beautiful responsive design with real hiking photos
- ⏱️ Track trip details: duration, distance, elevation gain
- 👥 Manage participant counts and trip availability
- 🔍 Filter and sort trips by various criteria

## Development

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

The app will open at `http://localhost:5173`

## Building

Build for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Technologies

- **Framework**: SvelteKit 5
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **State Management**: Svelte Stores

## Developing
